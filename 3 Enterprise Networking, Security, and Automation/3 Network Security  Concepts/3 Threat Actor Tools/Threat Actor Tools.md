### شرح القسم عن "Threat Actor Tools" (أدوات الجهات المهددة) بأبسط طريقة ممكنة

هذا القسم من كورس سيسكو عن الأمن السيبراني، يتكلم عن الأدوات اللي يستخدمها الهكرز (الجهات المهددة) عشان يخترقوا الأنظمة، وكمان أنواع الهجمات. هو يركز على إن الأدوات دي تطورت مع الوقت وأصبحت أسهل، ويمكن تستخدمها الطيبين (white hats) أو الشريرين (black hats). هقسمه لك عشان يكون سهل:

#### 1. **مقدمة عن أدوات الهجوم (Introduction to Attack Tools)**
   - عشان يستغل ثغرة أمنية (مشكلة في النظام)، الهكر لازم يكون عنده أداة أو طريقة.
   - مع الوقت، الأدوات دي أصبحت أكثر ذكاءً وتلقائية (مش محتاجة معرفة فنية كبيرة عشان تستخدمها). ده يعني إن أي حد يقدر يهاجم دلوقتي بسهولة أكبر.

#### 2. **تطور أدوات الأمن (Evolution of Security Tools)**
   - هنا جدول يصف أنواع أدوات الاختراق (penetration testing tools)، اللي تستخدم في اختبار الأمان. بعضها يستخدمها الطيبين عشان يحسنوا الأمان، وبعضها الشريرين عشان يهاجموا. الجدول مش كامل، لأن أدوات جديدة بتطلع كل يوم.
   - إليك الجدول باختصار:

     | نوع الأداة | وصفها | أمثلة |
     |------------|--------|--------|
     | **Password Crackers (كاسرات كلمات المرور)** | أدوات تكسر أو تسترد كلمات المرور بتجربة تخمينات كتير. | John the Ripper, Ophcrack, THC Hydra, Medusa. |
     | **Wireless Hacking Tools (أدوات اختراق الواي فاي)** | تستخدم عشان تخترق شبكات واي فاي وتكتشف ثغرات. | Aircrack-ng, Kismet, InSSIDer. |
     | **Network Scanning and Hacking Tools (أدوات مسح الشبكات)** | تبحث عن أجهزة ومنافذ مفتوحة في الشبكات. | Nmap, SuperScan, Angry IP Scanner. |
     | **Packet Crafting Tools (أدوات صنع الحزم)** | تصنع حزم بيانات مزيفة عشان تختبر الجدران النارية (firewalls). | Hping, Scapy, Netcat. |
     | **Packet Sniffers (أدوات شم الحزم)** | تلتقط وتحلل حركة البيانات في الشبكات. | Wireshark, Tcpdump, Ettercap. |
     | **Rootkit Detectors (كاشفات الروتكيت)** | تكشف البرامج الخفية اللي تخفي الهجمات. | AIDE, Netfilter. |
     | **Fuzzers (أدوات البحث عن ثغرات)** | تبحث عن ثغرات أمنية بتجربة إدخالات عشوائية. | Skipfish, Wapiti. |
     | **Forensic Tools (أدوات التحقيق الجنائي)** | تبحث عن أدلة في الحواسيب بعد هجوم. | Sleuth Kit, Maltego, Encase. |
     | **Debuggers (أدوات التصحيح)** | تستخدم عشان تحلل البرامج أو تصنع هجمات. | GDB, WinDbg, IDA Pro. |
     | **Hacking Operating Systems (أنظمة تشغيل للهكر)** | أنظمة جاهزة مليانة أدوات هكر. | Kali Linux, BackBox Linux. |
     | **Encryption Tools (أدوات التشفير)** | تشفر البيانات عشان تحميها من الوصول غير المصرح. | VeraCrypt, OpenSSH, Tor. |
     | **Vulnerability Exploitation Tools (أدوات استغلال الثغرات)** | تختبر إذا النظام عرضة لهجوم. | Metasploit, Sqlmap, Social Engineer Toolkit. |
     | **Vulnerability Scanners (ماسحات الثغرات)** | تمسح الشبكات بحثاً عن منافذ مفتوحة أو ثغرات معروفة. | Nessus, OpenVAS, SAINT. |

#### 3. **أنواع الهجمات (Attack Types)**
   - هنا وصف لأنواع الهجمات الشائعة اللي يستخدمها الهكرز باستخدام الأدوات دي:
     - ال**Eavesdropping Attack (هجوم التنصت)**: الهكر يلتقط ويسمع حركة البيانات في الشبكة (زي الشم أو السمع السري).
     - ال**Data Modification Attack (هجوم تعديل البيانات)**: بعد ما يلتقط البيانات، يغيرها بدون ما اللي بيرسل أو يستقبل يعرف.
     - ال**IP Address Spoofing Attack (هجوم تزييف عنوان IP)**: يصنع حزم IP تبدو وكأنها من عنوان داخلي صالح، عشان يخدع النظام.
     - ال**Password-Based Attacks (هجمات بناءً على كلمات المرور)**: إذا اكتشف حساب صالح، يقدر يسرق بيانات، يغير إعدادات، أو يحذف معلومات.
     - ال**Denial of Service Attack (DoS - هجوم رفض الخدمة)**: يغرق النظام أو الشبكة بحركة كتير عشان يوقفها أو يبطئها، فالناس الشرعيين مش يقدروا يستخدموها.
     - ال**Man-in-the-Middle Attack (هجوم الرجل في الوسط)**: الهكر يقف بين المرسل والمستقبل، يراقب، يلتقط، ويسيطر على الاتصال بدون ما حد يعرف.
     - ال**Compromised-Key Attack (هجوم المفتاح المخترق)**: إذا سرق مفتاح سري، يقدر يدخل على اتصالات آمنة بدون علم الآخرين.
     - ال**Sniffer Attack (هجوم الشم)**: أداة تقرأ وتراقب حركة البيانات، وإذا مش مشفرة، تشوف كل التفاصيل داخلها.

في النهاية، القسم ده يوضح إن الأدوات دي قوية ومتاحة، فالأمن السيبراني مهم عشان نحمي أنفسنا. لو عايز تفاصيل أكتر عن أداة معينة أو مثال، قولي!
