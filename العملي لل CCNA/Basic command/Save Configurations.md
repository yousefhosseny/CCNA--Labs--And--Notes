### 1. **حفظ التغييرات اللي عملتها في ال Router 
```
en
    copy running-config startup-config
  enter  بعد كده دوس 
```
(ده بيحفظ الإعدادات عشان تشتغل بعد إعادة التشغيل).


### 2. [[3 حفظ الإعدادات في ملف نصي (Capture Configuration to a Text File)]]

### 3. لو عملت اعدادات غلط قبل م تحفظ الاعدادات
- ممكن تمسح الأوامر الغلط واحد واحد يدويًا.
  - أو تعمل إعادة تشغيل للجهاز باستخدام:
    ```
    reload
    ```
    (تحذير: ده هيوقف الجهاز مؤقتًا، يعني الشبكة هتتوقف للحظات).

### 4. لو عملت اعدادات غلط بعد م حفظت الاعدادات
- امسح ملف الـ startup-config باستخدام:
    ```
    erase startup-config
    ```
  - بعد كده، اعمل `reload` عشان تمسح الـ running-config من الـ RAM.
  - (ملاحظة: بعد المسح، الجهاز هيرجع للإعدادات الافتراضية).
