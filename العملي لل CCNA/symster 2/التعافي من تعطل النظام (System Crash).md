### **خطوات التعافي من تعطل النظام (System Crash):**

1. **الاتصال عبر منفذ Console**
    
    - وصل جهاز الكمبيوتر بالسويتش باستخدام **كابل Console (RJ45 أو USB حسب الموديل)**.
        
    - افتح برنامج طرفية (زي PuTTY أو Tera Term) واضبط الإعدادات:
        
        - **9600** baud
            
        - 8 data bits
            
        - no parity
            
        - 1 stop bit
            
        - no flow control
            
2. **إيقاف تشغيل السويتش**
    
    - افصل كابل الطاقة من المحول/السويتش.
        
3. **الدخول إلى وضع Boot Loader**
    
    - أعد توصيل كابل الطاقة.
        
    - خلال **15 ثانية الأولى** اضغط مع الاستمرار على زر **Mode** (الموجود في واجهة السويتش) لحد ما لمبة **SYST** تتحول:
        
        - من أخضر → كهرماني → أخضر تاني.
            
    - بعدها هتشوف على شاشة الـ Console موجه الأوامر:
        
        ```
        switch:
        ```
        
4. **التأكد من الملفات الموجودة في الفلاش**
    
    - استخدم أمر:
        
        ```
        dir flash:
        ```
        
        ده هيعرض محتويات الفلاش، وتشوف إذا كان ملف الـ **IOS** موجود أو لأ.
        
5. **في حالة فقدان/تلف ملف IOS**
    
    - لو الملف مش موجود أو تالف:
        
        - وصّل السويتش بجهاز كمبيوتر فيه **TFTP server**.
            
        - انسخ ملف الـ IOS من الكمبيوتر للفلاش باستخدام:
            
            ```
            copy tftp: flash:
            ```
            
            وهتكتب IP السيرفر، اسم الملف، والمكان اللي هيتخزن فيه.
            
6. **تعيين ملف الإقلاع (Boot)**
    
    - بعد نقل ملف الـ IOS، لازم تحدد للسويتش الملف اللي يقلع منه:
        
        ```
        set BOOT flash:اسم_ملف_IOS.bin
        ```
        
7. **إعادة تحميل النظام**
    
    - اكتب:
        
        ```
        boot
        ```
        
        علشان يعمل تشغيل بالـ IOS الجديد.
        
8. **استعادة كلمة المرور (اختياري لو محتاج)**
    
    - من وضع Boot Loader، تقدر تعمل تخطي للـ startup-config وتشيل الباسورد:
        
        ```
        rename flash:config.text flash:config.old
        boot
        ```
        
    - بعد ما يشتغل السويتش، تدخل على الـ Privileged Mode وتسترجع الإعدادات من `config.old`.
