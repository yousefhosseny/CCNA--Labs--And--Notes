يعني مثلا عايز اوصل للسيرفر اللي في الشبكه دي بس ومش عايز اوصل لاي جهاز اخر

![[Pasted image 20250908222133.png]]



- **مثال على راوتر Branch**:
  - عشان توصل لسيرفر معين:
    - **لـ IPv4**:
      ```bash
      ip route 209.165.200.238 255.255.255.255 198.51.100.2 s0/1/0
      ```
      - **التفسير**:
        - `209.165.200.238`: ده هو ال IP بتاع الجهاز اللي انت عايز تروحله في الشبكه
        - `255.255.255.255`: ده دايما بيستخدمه ال Subnet ده في الطريقه دي او استخدم /32 
        - `198.51.100.2`: ده عنوان الـ Next-Hop  زي امر الراوتنج عادي بقي 
        - ال`s0/1/0` :  ال interface اللي هيخرج منه
