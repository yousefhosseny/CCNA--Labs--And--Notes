يعني مثلا عايز اوصل للسيرفر اللي في الشبكه دي بس ومش عايز اوصل لاي جهاز اخر

![[Pasted image 20250908222133.png]]

 - **لـ IPv6**:
      ```bash
      ipv6 route 2001:db8:acad:2::238/128 2001:db8:acad:1::2 s0/1/0
      ```
 - **التفسير**:
        - ال`2001:db8:acad:2::238/128`: ده هو ال IP بتاع الجهاز اللي انت عايز تروحله في الشبكه
        - `/128` لاحظ استخدم /128 بعد ال ip علي طول وهتستخدمها في الطريقه دي علي طول ثابته يعني ال /128
        - ال`2001:db8:acad:1::2`: ده عنوان الـ Next-Hop  زي امر الراوتنج عادي بقي.
        -   ال`s0/1/0` :  ال interface اللي هيخرج منه
