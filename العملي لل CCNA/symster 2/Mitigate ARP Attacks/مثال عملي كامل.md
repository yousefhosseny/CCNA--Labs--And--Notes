
# 6) سيناريو تكوين سريع (مجمّع) — مثال عملي كامل (VLAN 10)

```
! global
Switch(config)# ip dhcp snooping
Switch(config)# ip dhcp snooping vlan 10

Switch(config)# ip arp inspection vlan 10

! uplink to router (trusted)
Switch(config)# interface GigabitEthernet1/0/1
Switch(config-if)# switchport trunk encapsulation dot1q
Switch(config-if)# switchport mode trunk
Switch(config-if)# ip dhcp snooping trust
Switch(config-if)# ip arp inspection trust

! access port (PC)
Switch(config)# interface GigabitEthernet1/0/10
Switch(config-if)# switchport mode access
Switch(config-if)# switchport access vlan 10
Switch(config-if)# ip dhcp snooping limit rate 10
Switch(config-if)# ip arp inspection limit rate 15
```

> ملاحظة: الصيغة العامة للأوامر كما في الأمثلة أعلاه، لكن المنصة وإصدار IOS/NX-OS ممكن يغيّر بعض التفاصيل؛ راجع دليل الجهاز عند التطبيق.
