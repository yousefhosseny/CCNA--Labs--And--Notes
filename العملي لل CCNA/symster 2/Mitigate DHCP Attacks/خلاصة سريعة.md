
# خلاصة سريعة — Checklist للتطبيق على S1

-  `ip dhcp snooping` (global)
    
-  `ip dhcp snooping vlan 5,10,50-52`
    
-  `interface <server-port>` → `ip dhcp snooping trust`
    
-  `interface range Fa0/5 - 24` → `ip dhcp snooping limit rate 6`
    
-  تحقق: `show ip dhcp snooping` و`show ip dhcp snooping binding`
    
-  (اختياري) فعّل DAI: `ip arp inspection vlan ...` بعد التحقق من الـbindings
