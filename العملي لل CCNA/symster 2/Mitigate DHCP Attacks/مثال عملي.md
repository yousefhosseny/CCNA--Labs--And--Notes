
# مثال عملي على S1 (بافتراض DHCP server عند 192.168.10.10، ومنافذ F0/5–F0/24 هي untrusted)

```text
! شغّل DHCP snooping عام
S1(config)# ip dhcp snooping

! فعّل على فيلانز المطلوبة
S1(config)# ip dhcp snooping vlan 5,10,50-52

! علامة البورت المتصل بالسيرفر على سبيل المثال Fa0/2
S1(config)# interface FastEthernet0/2
S1(config-if)# ip dhcp snooping trust
S1(config-if)# exit

! على مجموعة البورتات اللي هم access وهنعتبرهم untrusted — نحدد معدل
S1(config)# interface range FastEthernet0/5 - 24
S1(config-if-range)# ip dhcp snooping limit rate 6
S1(config-if-range)# exit

! احفظ الإعدادات
S1# write memory
```

مهم: لو السيرفر مش مباشر على السويتش ولكن عبر راوتر (DHCP relay / ip helper-address) فالبورت اللي رايح للراوتر أو uplink للسويتش التاني لازم يكون **trusted** لأن الردود هترجع منه.
