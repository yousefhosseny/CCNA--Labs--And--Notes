
- لحذف pool بالكامل:
    
    ```
    Router(config)# no ip dhcp pool OFFICE
    ```
    
- لمسح binding معين:
    
    ```
    Router# clear ip dhcp binding 192.168.1.50
    ```
    
- لمسح جدول التعارضات:
    
    ```
    Router# clear ip dhcp conflict *
    ```
    

(استخدم `clear` بحذر في بيئة إنتاج لأن العملاء قد يحتاجوا لتجديد الـ IP بعده). ([firewall.cx](https://www.firewall.cx/cisco/cisco-switches/cisco-switch-router-dhcp-server-conflicts.html?utm_source=chatgpt.com "Resolving Cisco Switch & Router 'DHCP Server Pool ..."))
