بعد ما تعمل الإعداد، لازم تتأكد إن الراوتر فعلاً أخذ IP:

```bash
Router# show ip interface brief
```

هتلاقي مثلاً:

```
Interface              IP-Address      OK? Method Status                Protocol
GigabitEthernet0/0/1   209.165.201.12  YES DHCP   up                    up
```

- هنا باين إن الواجهة خدت IP `209.165.201.12` عن طريق الـ DHCP.
    

ممكن كمان تستخدم:

```bash
Router# show running-config
```

هتلاحظ إن الواجهة مكتوب عليها:

```
ip address dhcp
```
