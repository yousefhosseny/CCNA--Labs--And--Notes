
### 2. **أوامر التشخيص الشائعة (Common Troubleshooting Commands)**
الأوامر دي بتساعدك تشخّص مشاكل الشبكة:
- **`ping`**:
  - بيختبر الاتصال على المستوى 3 (Layer 3).
  - الـ Extended Ping يعطيك خيارات إضافية زي اختيار واجهة معينة.
- **`traceroute`**:
  - بيوريك المسار اللي الحزمة بتمشي فيه للوجهة باستخدام رسائل ICMP.
- **`show ip route`**:
  - بيوريك جدول التوجيه عشان تشوف المسارات الموجودة للوجهة.
- **`show ip interface brief`**:
  - بيوريك حالة واجهات الراوتر (شغالة ولا لا، وعناوين IP بتاعتها).
- **`show cdp neighbors`**:
  - بيوريك الأجهزة المتصلة مباشرة (Cisco) وبتأكد من الاتصال على المستوى 1 و2.
