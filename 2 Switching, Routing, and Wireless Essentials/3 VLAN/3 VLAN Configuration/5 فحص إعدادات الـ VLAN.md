### 5. **فحص إعدادات الـ VLAN**
- الأمر **show vlan** بيظهر تفاصيل الـ VLANs:
  - **show vlan brief**: بيظهر قايمة بالـ VLANs وحالتهم والمنافذ المرتبطة بيهم.
  - **show vlan id 20**: بيظهر تفاصيل VLAN معين (مثل VLAN 20).
  - **show vlan name student**: بيظهر تفاصيل VLAN بالاسم.
  - **show vlan summary**: بيظهر ملخص عن الـ VLANs.
- الأمر **show interfaces fa0/18 switchport** بيظهر إعدادات المنفذ (زي الـ VLAN المرتبط).
