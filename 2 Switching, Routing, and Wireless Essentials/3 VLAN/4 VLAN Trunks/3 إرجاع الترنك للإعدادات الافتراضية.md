### 3. **إرجاع الترنك للإعدادات الافتراضية**
- لو عايز ترجّع الترنك للوضع الافتراضي:
  1. استخدم أمر `no` لإلغاء الإعدادات:
     ```
     Switch(config)# interface fa0/1
     Switch(config-if)# no switchport trunk native vlan
     Switch(config-if)# no switchport trunk allowed vlan
     Switch(config-if)# end
     ```
     - ده هيرجّع الـ Native VLAN لـ VLAN 1، وهيسمح لكل الـ VLANs بالمرور.
  2. تحقق من الإعدادات الافتراضية باستخدام:
     ```
     show interfaces fa0/1 switchport
     ```

- لو عايز تحول المنفذ من ترنك لـ Access Port:
  ```
  Switch(config)# interface fa0/1
  Switch(config-if)# switchport mode access
  Switch(config-if)# end
  ```
  - ده هيخلي المنفذ Access Port ومش هيبقى ترنك، وهيرتبط بـ VLAN 1 افتراضيًا.
