
### 3. **الخطوة 1: اختيار Root Bridge**
- كل السويتشات بتشارك في اختيار الـ Root Bridge.
- كل سويتش بيبعت BPDUs كل ثانيتين، وفيها:
  - BID بتاعه (هويته).
  - Root ID (BID بتاع السويتش اللي هو شايفه الـ Root Bridge).
- في البداية، كل سويتش بيفترض إنه الـ Root Bridge، بس مع تبادل الـ BPDUs، بيختاروا السويتش اللي عنده أقل BID.
- لو فيه تعادل في الأولوية (Priority)، السويتش اللي عنده أقل MAC Address بيبقى الـ Root Bridge.
- **مثال**:
  - لو S1 عنده Priority 24577، وS2 وS3 عندهم Priority 32769، S1 هيبقى الـ Root Bridge لأن أولويته أقل.
