### 6. **فحص إعدادات Router-on-a-Stick**
- استخدم الأوامر دي للتأكد من إعدادات الراوتر والسويتش:
  - على الراوتر:
    ```
    show ip route
    ```
    (بيظهر جدول الروتينج، لازم تشوف الشبكات 192.168.10.0, 192.168.20.0, 192.168.99.0).
    ```
    show ip interface brief
    ```
    (بيظهر حالة المنافذ والـ Subinterfaces).
    ```
    show interfaces
    ```
    (بيظهر تفاصيل المنافذ).
  - على السويتش:
    ```
    show interfaces trunk
    ```
    (بيظهر إعدادات الترنك، زي الـ Native VLAN والـ VLANs المسموح بيها).
