
### 3. **إعدادات سويتش S2**
- نفس إعدادات S1 بالظبط (إنشاء VLANs، واجهة إدارة، منافذ أكسس، ومنافذ ترنك). بس ممكن نربط أجهزة تانية (زي PC2) بـ VLAN 20:
  ```
  S2(config)# interface fa0/18
  S2(config-if)# switchport mode access
  S2(config-if)# switchport access vlan 20
  S2(config-if)# end
  ```
