
### 4. **فحص الاتصال**
- بعد الإعداد، نقدر نختبر الاتصال بين PC1 (VLAN 10) وPC2 (VLAN 20):
  1. من PC1، افتح موجه الأوامر وتأكد من إعدادات الـ IP:
     ```
     ipconfig
     ```
     (هتظهر عنوان IP زي 192.168.10.x لـ PC1 و192.168.20.x لـ PC2).
  2. اعمل ping من PC1 لـ PC2:
     ```
     ping 192.168.20.x
     ```
     لو الـ ping نجح، يبقى الـ Inter-VLAN Routing شغال.
