
### 4. **مشكلة: منفذ الترنك**
- **السبب**: المنفذ اللي بيربط السويتش بالراوتر مش معمول إعداده كترنك، أو فيه مشكلة في إعدادات الترنك (زي VLANs غير مسموح بيها).
- **الحل**:
  1. تأكد إن المنفذ ترنك:
     ```
     S1(config)# interface fa0/5
     S1(config-if)# switchport mode trunk
     S1(config-if)# switchport trunk allowed vlan 10,20,99
     S1(config-if)# end
     ```
- **التحقق**:
  ```
  show interfaces trunk
  show running-config interface fa0/5
  ```
  (لو المنفذ مش ظاهر في `show interfaces trunk`، يبقى مش معمول إعداده كترنك).
