
### 5. **مشكلة: منفذ الأكسس**
- **السبب**: المنفذ اللي متصل بالجهاز (زي PC1) مش مربوط بالـ VLAN الصحيح، أو مش مفعّل كأكسس، أو إعدادات الـ IP في الجهاز غلط.
- **الأعراض**: الجهاز عنده IP وSubnet وDefault Gateway صح، بس مابيعرفش يعمل ping للـ Default Gateway.
- **الحل**:
  1. تأكد إن المنفذ مربوط بالـ VLAN الصحيح:
     ```
     S1(config)# interface fa0/6
     S1(config-if)# switchport mode access
     S1(config-if)# switchport access vlan 10
     S1(config-if)# end
     ```
  2. من الجهاز، تأكد من إعدادات الـ IP:
     ```
     ipconfig
     ```
     (تأكد إن الـ IP زي 192.168.10.x والـ Default Gateway هو 192.168.10.1).
- **التحقق**:
  ```
  show vlan brief
  show interfaces fa0/6 switchport
  show running-config interface fa0/6
  ```
