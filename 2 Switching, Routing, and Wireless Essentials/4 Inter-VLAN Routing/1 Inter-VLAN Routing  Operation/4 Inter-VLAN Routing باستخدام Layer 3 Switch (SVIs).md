
### 4. **Inter-VLAN Routing باستخدام Layer 3 Switch (SVIs)**
- **إزاي بتشتغل؟**
  - الـ Layer 3 Switch (أو Multilayer Switch) بيقدر يعمل روتينج بين VLANs باستخدام **Switched Virtual Interfaces (SVIs)**.
  - الـ SVI هي واجهة افتراضية بتتربط بـ VLAN معين في السويتش. بتعمل زي واجهة الراوتر، بس داخل السويتش نفسه.
  - كل SVI بيتربط بـ VLAN وبياخد عنوان IP يناسب الـ Subnet بتاع الـ VLAN.
- **المميزات**:
  - **سرعة عالية**: الروتينج بيتم داخل السويتش باستخدام الأجهزة (Hardware)، فأسرع بكتير من Router-on-a-Stick.
  - **ما بتحتاجش روابط خارجية**: كل حاجة بتتم داخل السويتش.
  - **تدعم شبكات كبيرة**: ينفع تستخدم EtherChannels (تجميع منافذ) عشان تزود السرعة.
  - **زمن تأخير أقل**: البيانات مابتطلعش برا السويتش.
  - **شائع في الشبكات الكبيرة**: بيستخدم في الشركات والمؤسسات الكبيرة.
- **العيب**:
  - الـ Layer 3 Switch أغلى من الراوتر العادي.
