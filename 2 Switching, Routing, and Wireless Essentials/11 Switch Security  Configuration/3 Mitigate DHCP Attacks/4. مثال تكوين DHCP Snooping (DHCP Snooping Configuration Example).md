#### 4. **مثال تكوين DHCP Snooping (DHCP Snooping Configuration Example)**
   - افترض شبكة مع سويتش S1، DHCP server على 192.168.10.10، ومنافذ F0/5 إلى F0/24 untrusted.
     - أوامر على S1:
       - `ip dhcp snooping` (فعل عام).
       - على الـinterface للـserver: `ip dhcp snooping trust`.
       - على untrusted ports: `ip dhcp snooping limit rate 6`.
       - `ip dhcp snooping vlan 5,10,50,51,52` (فعل على VLANs دي).
   - **ملاحظة**: DHCP Snooping مطلوب أيضاً لميزة Dynamic ARP Inspection (DAI) في أقسام لاحقة.
