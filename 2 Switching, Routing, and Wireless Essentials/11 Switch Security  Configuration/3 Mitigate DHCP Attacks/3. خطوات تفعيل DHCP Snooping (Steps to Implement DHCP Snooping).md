#### 3. **خطوات تفعيل DHCP Snooping (Steps to Implement DHCP Snooping)**
   - الخطوات الأربعة البسيطة في Cisco IOS:
     1. **فعلها عام**: `ip dhcp snooping` (في وضع global config).
     2. **حدد المنافذ الموثوقة**: على الـports اللي متصلة بـDHCP server أو سويتشات أخرى: `ip dhcp snooping trust` (في وضع interface config).
     3. **حد سرعة الرسائل على untrusted ports**: عشان ما يغرقش: `ip dhcp snooping limit rate <عدد packets per second>` (مثل 6).
     4. **فعلها على VLANs**: `ip dhcp snooping vlan <رقم أو نطاق>` (مثل 5,10,50-52).

   - **جدول بسيط للخطوات**:
     | الخطوة | الوصف المبسط | الأمر الرئيسي |
     |--------|---------------|----------------|
     | 1     | فعل DHCP snooping عام | `ip dhcp snooping` |
     | 2     | حدد ports موثوقة | `ip dhcp snooping trust` |
     | 3     | حد سرعة على untrusted | `ip dhcp snooping limit rate <عدد>` |
     | 4     | فعل على VLANs | `ip dhcp snooping vlan <أرقام>` |
