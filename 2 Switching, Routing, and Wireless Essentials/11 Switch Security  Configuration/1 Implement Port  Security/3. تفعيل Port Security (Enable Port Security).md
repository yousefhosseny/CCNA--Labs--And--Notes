#### 3. **تفعيل Port Security (Enable Port Security)**
   - **خطوات بسيطة**:
     - أولاً، تأكد إن الـport في وضع access (مش trunk افتراضياً). أمر: `switchport mode access`.
     - ثم فعلها: `switchport port-security`.
   - **ملاحظة**: لو الـport trunk، الأمان ده خارج نطاق الكورس. ولو وصل أكتر من جهاز واحد، الـport هيتقفل تلقائياً (error-disabled).
   - **تحقق**: أمر `show port-security interface FastEthernet 0/1` يظهر إعدادات زي: مفعل، عدد MAC max=1، وضع الانتهاك=shutdown.
