#### 5. **الشيخوخة (Port Security Aging)**
   - **الفكرة**: لو عايز تمسح MAC addresses الآمنة بعد فترة، عشان تحديث أو أمان.
     - أنواع: 
       - Absolute: تمسح بعد وقت محدد (سواء نشط أو لا).
       - Inactivity: تمسح لو الـMAC غير نشط لفترة.
   - **أمر**: `switchport port-security aging {static | time <دقائق> | type {absolute | inactivity}}`.
   - **مثال**: `switchport port-security aging type inactivity` و `switchport port-security aging time 10` (تمسح بعد 10 دقائق عدم نشاط).
   - **فائدة**: ينظف الجدول بدون تدخل يدوي.
