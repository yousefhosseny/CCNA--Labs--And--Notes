#### 8. **التحقق من الإعدادات (Verify Port Security)**
   - **أوامر أساسية**:
     - ء  `show port-security`: يظهر إعدادات كل الـports (مثل max، current count).
     - ء `show port-security interface <port>`: تفاصيل port واحد (مثل enabled، violation mode).
     -  ء `show port-security address`: كل MAC addresses الآمنة على السويتش.
   - **مثال**: لو كل ports مفعلة، وما فيش أجهزة متصلة، current count = 0.

#### ملاحظات عامة:
- **لماذا مهم؟** Layer 2 هجمات سهلة، لكن الحلول دي بسيطة وفعالة تحول السويتش لأكثر أمان.
- **نصيحة**: ابدأ دائماً بإغلاق unused ports، ثم فعل port security على access ports، وحدد MAC limits حسب احتياجك (مثل 1 لجهاز واحد).
- لو عايز تمارس، استخدم Packet Tracer أو جهاز سيسكو حقيقي.
