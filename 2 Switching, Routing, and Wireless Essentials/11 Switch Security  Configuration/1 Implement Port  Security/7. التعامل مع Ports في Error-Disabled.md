
#### 7. **التعامل مع Ports في Error-Disabled**
   - **المشكلة**: لو انتهاك في وضع shutdown، الـport يتقفل، LED off، وترسل رسائل console.
   - **تحقق**: `show interfaces` (يظهر err-disabled)، `show port-security interface` (secure-shutdown).
   - **حل**: تحقق السبب (جهاز غير مصرح؟)، ثم `shutdown` ثم `no shutdown` عشان ترجع الـport.
   - **ملاحظة**: العداد يزيد كل مرة انتهاك.
