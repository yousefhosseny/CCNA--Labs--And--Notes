
#### 6. **أوضاع الانتهاك (Port Security Violation Modes)**
   - لو جهاز غير مصرح اتصل (MAC مختلف)، يحصل انتهاك، والسويتش يرد حسب الوضع:
     - ء **Shutdown (افتراضي)**: يقفل الـport تماماً (error-disabled)، يرسل رسالة syslog، ويطفي LED. تحتاج `shutdown` ثم `no shutdown` عشان ترجعه.
     - ء **Restrict**: يرمي الـpackets الغريبة، يزود عداد الانتهاك، يرسل syslog، لكن الـport يفضل مفتوح.
     - ء **Protect**: يرمي الـpackets الغريبة بدون syslog أو عداد (أقل أمان).
   - **أمر تغيير**: `switchport port-security violation {shutdown | restrict | protect}`.
   - **جدول بسيط**:
     | وضع       | رد السويتش                          |
     |-----------|-------------------------------------|
     | Shutdown | يقفل الـport، رسالة، عداد يزيد.   |
     | Restrict | يرمي packets، رسالة، عداد يزيد.  |
     | Protect  | يرمي packets فقط، بدون رسائل.    |
