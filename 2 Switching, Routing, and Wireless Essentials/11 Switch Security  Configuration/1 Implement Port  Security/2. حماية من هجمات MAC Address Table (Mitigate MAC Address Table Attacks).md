
#### 2. **حماية من هجمات MAC Address Table (Mitigate MAC Address Table Attacks)**
   - **المشكلة**: الهكرز ممكن يغرق جدول MAC addresses في السويتش بكم هائل من العناوين الوهمية، وده يسبب overflow ويفتح باب للهجمات.
   - **الحل الأفضل**: فعل **Port Security**، اللي يحد من عدد MAC addresses المسموحة على كل port.
     - السويتش يقارن عنوان MAC للجهاز اللي متصل مع قائمة آمنة (إما محددة يدوياً أو متعلمة تلقائياً).
     - لو حددت عدد MAC واحد بس، ده يمنع أي جهاز تاني من الاتصال، ويسيطر على الوصول غير المصرح.
