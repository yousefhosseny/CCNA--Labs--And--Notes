
#### 4. **تحديد وتعلم MAC Addresses (Limit and Learn MAC Addresses)**
   - **حدد العدد الأقصى**: `switchport port-security maximum <عدد>` (افتراضي 1، ممكن يصل لـ8192 حسب السويتش).
   - **طرق تعلم MAC** (السويتش يتعلم عناوين MAC الآمنة بـ3 طرق):
     1. **يدوي (Manually)**: أنت تحدد العنوان بنفسك: `switchport port-security mac-address <MAC-address>`.
     2. **تلقائي (Dynamically)**: السويتش يتعلم تلقائياً من الجهاز المتصل لما تفعل `switchport port-security`، لكن لو reboot، هيفقد الذاكرة.
     3. **تلقائي مع Sticky**: يتعلم تلقائياً ويحفظ في الـrunning config: `switchport port-security mac-address sticky`. (ده يحفظ في NVRAM لو حفظت الـconfig).
   - **مثال كامل**: لو عايز max=4، واحد يدوي، والباقي تلقائي: فعل الـport security، حدد maximum 4، أضف MAC يدوي، ثم sticky.
   - **تحقق**: `show port-security interface` و `show port-security address`.
