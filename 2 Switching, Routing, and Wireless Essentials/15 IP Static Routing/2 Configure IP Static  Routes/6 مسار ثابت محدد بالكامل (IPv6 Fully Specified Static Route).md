
### 6. **مسار ثابت محدد بالكامل (IPv6 Fully Specified Static Route)**
- نفس الفكرة لـ IPv6، بس لازم تستخدمه لو الـ Next-Hop هو عنوان **Link-Local** (زي FE80::/10).
- **السبب**:
  - عناوين Link-Local مش موجودة في جدول التوجيه، وهي صالحة بس على رابط معين، فبتحتاج تحدد الواجهة عشان تتجنب اللبس.
- **مثال**:
  ```bash
  ipv6 route 2001:db8:acad:1::/64 s0/1/0 FE80::2
  ```
- **التفسير**:
  - بتحدد الشبكة الوجهة (2001:db8:acad:1::/64)، واجهة الخروج (s0/1/0)، وعنوان Link-Local للـ Next-Hop (FE80::2).
  - الراوتر بيضيف المسار ده في جدول التوجيه مع ذكر الواجهة والـ Next-Hop.
