
### 1. **مسار ثابت باستخدام عنوان الـ Next-Hop (IPv4 Next-Hop Static Route)**
- في النوع ده، بتحدد بس **عنوان IP للراوتر التالي (Next-Hop)**، والراوتر هيستنتج واجهة الخروج لوحده.
- **مثال على R1**:
  ```bash
  ip route 172.16.1.0 255.255.255.0 172.16.2.2
  ip route 192.168.1.0 255.255.255.0 172.16.2.2
  ip route 192.168.2.0 255.255.255.0 172.16.2.2
  ```
- **التفسير**:
  - كل أمر بيحدد شبكة وجهة (مثل 172.16.1.0/24) وقناع الشبكة (255.255.255.0) وعنوان الـ Next-Hop (172.16.2.2، وهو عنوان R2).
  - الراوتر بيضيف المسارات دي في جدول التوجيه، وهيبعت أي حزمة للشبكات دي عن طريق R2.
