
### 5. **مسار ثابت محدد بالكامل (IPv4 Fully Specified Static Route)**
- هنا بتحدد **واجهة الخروج وعنوان الـ Next-Hop** مع بعض.
- بيُستخدم لما الواجهة تكون متعددة الوصول (Multi-Access) زي الإيثرنت، عشان توضح الـ Next-Hop.
- **مثال**:
  - لو عايز تعمل مسار على واجهة إيثرنت، لازم تضيف عنوان الـ Next-Hop مع الواجهة.
