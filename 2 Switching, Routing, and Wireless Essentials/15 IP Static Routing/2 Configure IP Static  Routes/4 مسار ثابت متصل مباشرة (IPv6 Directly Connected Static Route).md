
### 4. **مسار ثابت متصل مباشرة (IPv6 Directly Connected Static Route)**
- نفس الفكرة لـ IPv6، بس بتحدد واجهة الخروج.
- **مثال على R1**:
  ```bash
  ipv6 route 2001:db8:acad:1::/64 s0/1/0
  ipv6 route 2001:db8:cafe:1::/64 s0/1/0
  ipv6 route 2001:db8:cafe:2::/64 s0/1/0
  ```
- **التفسير**:
  - بتحدد الشبكة الوجهة (مثل 2001:db8:acad:1::/64) وواجهة الخروج (s0/1/0).
  - زي IPv4، الطريقة دي بتُستخدم مع واجهات تسلسلية نقطة-لنقطة بس.
