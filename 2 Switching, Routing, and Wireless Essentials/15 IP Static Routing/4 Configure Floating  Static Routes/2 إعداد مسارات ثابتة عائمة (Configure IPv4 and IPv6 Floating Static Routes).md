
### 2. **إعداد مسارات ثابتة عائمة (Configure IPv4 and IPv6 Floating Static Routes)**
- **لـ IPv4**:
  - أمر إعداد المسار الافتراضي العادي:
    ```bash
    ip route 0.0.0.0 0.0.0.0 172.16.2.2
    ```
    - ده مسار افتراضي عادي (AD = 1) لتوجيه الحزم لـ R2.
  - أمر إعداد المسار العائم:
    ```bash
    ip route 0.0.0.0 0.0.0.0 10.10.10.2 5
    ```
    - ده مسار احتياطي (AD = 5) لتوجيه الحزم لـ R3 لو المسار الأساسي فشل.

- **لـ IPv6**:
  - أمر إعداد المسار الافتراضي العادي:
    ```bash
    ipv6 route ::/0 2001:db8:acad:2::2
    ```
    - مسار افتراضي (AD = 1) لتوجيه الحزم لـ R2.
  - أمر إعداد المسار العائم:
    ```bash
    ipv6 route ::/0 2001:db8:feed:10::2 5
    ```
    - مسار احتياطي (AD = 5) لتوجيه الحزم لـ R3 لو المسار الأساسي فشل.

- **ملاحظة**:
  - لما تتفرج على جدول التوجيه باستخدام `show ip route` أو `show ipv6 route`، المسار الأساسي (اللي AD = 1) بس هو اللي هيظهر في الجدول طالما شغال.
  - المسار العائم (AD = 5) مش هيظهر في الجدول إلا لو المسار الأساسي اختفى.
