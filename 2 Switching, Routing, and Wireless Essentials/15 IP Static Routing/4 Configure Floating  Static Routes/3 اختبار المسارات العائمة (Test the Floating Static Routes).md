
### 3. **اختبار المسارات العائمة (Test the Floating Static Routes)**
- لو الراوتر R2 (اللي عليه المسار الأساسي) فشل (مثلًا لو واجهاته التسلسلية اتقفلت):
  - R1 هيولد رسايل **syslog** تقول إن الرابط مع R2 وقّف.
  - المسار العائم (اللي لـ R3) هيظهر في جدول التوجيه ويبتدي يُستخدم بدل المسار الأساسي.
- لما تشوف جدول التوجيه باستخدام `show ip route` أو `show ipv6 route`، هتلاقي المسار العائم مفعّل.
