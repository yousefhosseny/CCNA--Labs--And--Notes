
### 3. **أمر إعداد المسار الافتراضي لـ IPv6**
- الأمر:
  ```bash
  ipv6 route ::/0 {ipv6-address | exit-intf}
  ```
- **التفسير**:
  - `::/0`: يعني "كل الشبكات"، لأن الـ Prefix /0 بيطابق أي عنوان IPv6.
  - `ipv6-address`: عنوان الـ Next-Hop.
  - `exit-intf`: واجهة الخروج (اختياري).
- **مثال على R1**:
  ```bash
  ipv6 route ::/0 2001:db8:acad:2::2
  ```
  - أي حزمة IPv6 مالهاش مسار محدد هتترسل لـ R2 على عنوان 2001:db8:acad:2::2.
