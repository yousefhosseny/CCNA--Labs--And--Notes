
### 4. **التحقق من المسار الافتراضي (Verify a Default Static Route)**
- **لـ IPv4**:
  - الأمر:
    ```bash
    show ip route static
    ```
  - المخرجات هتوريك المسار الافتراضي مع رمز `*` جنب كود `S` (Static)، وده بيوضح إنه مرشح ليكون Gateway of Last Resort.
  - الـ Subnet Mask `/0` يعني إن مفيش بتات لازم تتطابق، فالمسار ده بيطابق كل الحزم لو مكانش فيه مسار أكثر تحديدًا.
- **لـ IPv6**:
  - الأمر:
    ```bash
    show ipv6 route static
    ```
  - المخرجات هتوريك المسار الافتراضي مع الـ Prefix `::/0`، وده بيطابق كل الحزم لو مكانش فيه مسار أكثر تحديدًا.
