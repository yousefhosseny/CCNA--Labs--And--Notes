
### 2. **أمر إعداد المسار الافتراضي لـ IPv4**
- الأمر:
  ```bash
  ip route 0.0.0.0 0.0.0.0 {ip-address | exit-intf}
  ```
- **التفسير**:
  - `0.0.0.0 0.0.0.0`: يعني "كل الشبكات" (Quad-Zero Route)، لأن القناع /0 بيطابق أي عنوان.
  - `ip-address`: عنوان الـ Next-Hop (الراوتر التالي).
  - `exit-intf`: واجهة الخروج (اختياري).
- **مثال على R1**:
  ```bash
  ip route 0.0.0.0 0.0.0.0 172.16.2.2
  ```
  - أي حزمة مالهاش مسار محدد هتترسل لـ R2 على عنوان 172.16.2.2.
