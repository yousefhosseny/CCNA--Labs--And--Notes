
### 2. **خيارات الـ Next-Hop**
عند إعداد مسار ثابت، لازم تحدد الـ **Next-Hop** (الوجهة التالية) بإحدى الطرق دي:
- ال**Next-Hop Route**: تحدد بس عنوان IP للراوتر التالي.
- ال**Directly Connected Static Route**: تحدد بس واجهة الخروج (Exit Interface).
- ال**Fully Specified Static Route**: تحدد عنوان IP للراوتر التالي وواجهة الخروج مع بعض.
