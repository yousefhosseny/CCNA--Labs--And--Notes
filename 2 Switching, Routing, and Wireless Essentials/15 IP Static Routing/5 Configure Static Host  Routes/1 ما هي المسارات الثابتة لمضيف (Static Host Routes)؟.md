
### 1. **ما هي المسارات الثابتة لمضيف (Static Host Routes)؟**
- دي مسارات ثابتة يدوية بتُستخدم عشان توجه الحزم لجهاز معين (مثل سيرفر) بدقة.
- **الخصائص**:
  - في IPv4: بتستخدم قناع شبكة **255.255.255.255** (أو /32)، يعني بتطابق عنوان IP واحد بس.
  - في IPv6: بتستخدم طول الـ Prefix **/128**، يعني بتطابق عنوان IPv6 واحد بس.
