
### 2. **إعداد مسارات ثابتة لمضيف (Configure Static Host Routes)**
- **مثال على راوتر Branch**:
  - عشان توصل لسيرفر معين:
    - **لـ IPv4**:
      ```bash
      ip route 209.165.200.238 255.255.255.255 198.51.100.2
      ```
      - **التفسير**:
        - `209.165.200.238`: عنوان IP للسيرفر (الوجهة).
        - `255.255.255.255`: قناع الشبكة (/32) يعني المسار ده لعنوان واحد بس.
        - `198.51.100.2`: عنوان الـ Next-Hop (الراوتر التالي اللي هيبعتله الحزمة).
    - **لـ IPv6**:
      ```bash
      ipv6 route 2001:db8:acad:2::238/128 2001:db8:acad:1::2
      ```
      - **التفسير**:
        - `2001:db8:acad:2::238/128`: عنوان IPv6 للسيرفر مع طول Prefix /128.
        - `2001:db8:acad:1::2`: عنوان الـ Next-Hop.
