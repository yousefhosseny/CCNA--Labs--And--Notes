
### **2. أمر الـ Boot System:**
- المحول يحاول التشغيل تلقائيًا باستخدام متغير يسمى **BOOT**.
- إذا لم يتم تعيين هذا المتغير، المحول يبحث عن أول ملف قابل للتنفيذ في الفلاش.
- يمكن تحديد ملف IOS معين باستخدام الأمر:
  ```
  boot system flash:/<path>/<IOS-filename>
  ```
  مثال: 
  ```
  boot system flash:/c2960-lanbasek9-mz.150-2.SE/c2960-lanbasek9-mz.150-2.SE.bin
  ```
  - هنا يتم تحديد مكان ملف IOS واسم الملف.
- للتحقق من إعدادات الـ Boot، استخدم الأمر:
  ```
  show boot
  ```
