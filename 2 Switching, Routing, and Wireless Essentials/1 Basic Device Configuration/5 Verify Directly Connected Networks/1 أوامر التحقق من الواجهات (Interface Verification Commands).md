
### **1. أوامر التحقق من الواجهات (Interface Verification Commands):**
هناك عدة أوامر تُستخدم للتحقق من حالة وإعدادات واجهات الراوتر:

- **`show ip interface brief` و `show ipv6 interface brief`:**
  - يعرضان ملخصًا لجميع الواجهات.
  - يُظهر عنوان IP (IPv4 أو IPv6) وحالتها (نشطة أو غير نشطة).
  - مثال: إذا كانت الحالة "up/up"، فهذا يعني أن الواجهة تعمل بشكل صحيح.

- **`show running-config interface interface-id`:**
  - يعرض الأوامر المطبقة على واجهة معينة (مثل عنوان IP أو وصف الواجهة).

- **`show ip route` و `show ipv6 route`:**
  - يعرضان جدول التوجيه (Routing Table) لـ IPv4 أو IPv6.
  - الواجهات النشطة تظهر في الجدول مع رمزين:
    - **C** (Connected): شبكة متصلة مباشرة.
    - **L** (Local): عنوان الواجهة نفسها.
  - في الإصدارات القديمة من IOS، يظهر رمز **C** فقط.
