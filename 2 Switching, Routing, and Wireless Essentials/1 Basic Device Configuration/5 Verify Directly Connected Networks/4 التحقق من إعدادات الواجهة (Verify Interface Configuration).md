
### **4. التحقق من إعدادات الواجهة (Verify Interface Configuration):**
- **الأمر `show running-config interface interface-id`:**
  - يُظهر الإعدادات المطبقة على واجهة معينة (مثل عنوان IP أو حالة التفعيل).
- **أوامر إضافية لتفاصيل أكثر:**
  - **`show interfaces`:** يعرض إحصائيات تدفق الحزم ومعلومات عن جميع الواجهات.
  - **`show ip interface` و `show ipv6 interface`:** يعرضان معلومات متعلقة بـ IPv4 وIPv6 للواجهات.
