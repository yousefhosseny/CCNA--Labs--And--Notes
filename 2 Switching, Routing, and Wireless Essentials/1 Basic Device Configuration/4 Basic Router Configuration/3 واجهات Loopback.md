### **3. واجهات Loopback:**
- **واجهة Loopback** هي واجهة منطقية (غير فيزيائية) داخل الراوتر.
- **الاستخدامات:**
  - اختبار عمليات التوجيه الداخلية.
  - محاكاة شبكات إضافية في بيئات التجربة (Labs).
  - ضمان وجود واجهة نشطة دائمًا (لا تتوقف طالما الراوتر يعمل).
- **إعداد واجهة Loopback:**
  - مثال:
    ```bash
    R1(config)# interface loopback 0
    R1(config-if)# ip address 10.0.0.1 255.255.255.0
    R1(config-if)# exit
    ```
  - عنوان IP لكل واجهة Loopback يجب أن يكون فريدًا.
- **ملاحظة:** تُستخدم غالبًا لمحاكاة اتصال بالإنترنت في بيئات التجربة.
