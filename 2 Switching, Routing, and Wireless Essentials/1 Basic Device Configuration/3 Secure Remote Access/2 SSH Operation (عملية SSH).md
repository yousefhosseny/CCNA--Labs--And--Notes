
### **2. SSH Operation (عملية SSH):**
- ء  **SSH** (Secure Shell) هو بروتوكول آمن يستخدم منفذ TCP 22.
- **الميزة:** يقوم بتشفير جميع البيانات (اسم المستخدم، كلمة المرور، والبيانات المرسلة)، مما يجعله آمنًا.
- **الاستخدام:** يُفضل استخدام SSH بدلاً من Telnet لإدارة الأجهزة عن بُعد.
- **مثال:** إذا تم اعتراض جلسة SSH باستخدام Wireshark، يمكن رؤية عنوان IP فقط، لكن اسم المستخدم وكلمة المرور مشفرة ولا يمكن قراءتها.
