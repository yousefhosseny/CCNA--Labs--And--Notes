
### **5. التحقق من عمل SSH:**
- من جهاز كمبيوتر، استخدم برنامج مثل **PuTTY** للاتصال بالمحول عبر SSH.
- مثال:
  - إذا كان عنوان IP لواجهة VLAN 99 على المحول هو **172.17.99.11**، والكمبيوتر لديه عنوان **172.17.99.21**:
    1. افتح PuTTY وأدخل عنوان IP للمحول (172.17.99.11).
    2. اختر SSH كبروتوكول.
    3. أدخل اسم المستخدم (مثل admin) وكلمة المرور (مثل ccna).
    4. إذا نجحت العملية، ستتصل بواجهة CLI الخاصة بالمحول.
