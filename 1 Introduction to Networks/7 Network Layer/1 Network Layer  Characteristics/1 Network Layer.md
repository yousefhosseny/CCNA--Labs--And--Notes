بختصار وبأبسط طريقة ممكنة، القسم ده بيشرح الـ **Network Layer** (الطبقة الشبكية) في نموذج الشبكات، وهي الطبقة المسؤولة ==عن نقل البيانات بين الأجهزة في الشبكة==. إليك الشرح المبسط:

### **الطبقة الشبكية (Network Layer):**
- **وظيفتها الأساسية**: ==توصيل البيانات بين الأجهزة النهائية== (مثل الكمبيوترات أو الهواتف) عبر الشبكة.
- **البروتوكولات الرئيسية**: 
  - ==**IPv4**== و ==**IPv6**==، وهما البروتوكولات المستخدمة لنقل البيانات.
- **العمليات الأساسية**:
  1. **Addressing (العنونة)**: إعطاء كل جهاز عنوان IP فريد للتعرف عليه.
  2. **Encapsulation (التغليف)**: وضع البيانات داخل حزمة (Packet) مع إضافة رأس (Header) يحتوي على معلومات مثل عنوان المصدر والوجهة.
  3. **Routing (التوجيه)**: تحديد أفضل مسار لنقل الحزمة عبر الشبكة.
  4. **De-encapsulation (فك التغليف)**: عند وصول الحزمة للوجهة، يتم فكها لاستخراج البيانات.
