
#### **2. Server Message Block (SMB)**
- **الفكرة**: SMB بروتوكول تاني لمشاركة الملفات، لكن بيشتغل بطريقة مختلفة. بيسمح للعميل بالوصول لموارد السيرفر (ملفات، طابعات، إلخ) كأنها موجودة على جهازك.
- **ازاي بيشتغل؟**
  - بيتم إنشاء اتصال طويل الأمد بين العميل والسيرفر.
  - بعد الاتصال، العميل بيقدر يستخدم الملفات أو الطابعات على السيرفر وكأنها على جهازه.
- **وظايفه**:
  1. بدء وتأكيد وإنهاء الاتصال.
  2. التحكم في الوصول للملفات والطابعات.
  3. إرسال أو استقبال رسائل بين الأجهزة.
- **يعني ايه؟**: زي لما تشارك فولدر على شبكة الشركة، وأي جهاز متصل بالشبكة يقدر يفتح الملفات دي كأنها على جهازه.
