
#### **1. File Transfer Protocol (FTP)**
- **الفكرة**: FTP هو بروتوكول يُستخدم لنقل الملفات بين جهاز العميل (الكمبيوتر بتاعك) والسيرفر (جهاز الخادم).
- **ازاي بيشتغل؟**
  1. **الاتصال الأول (التحكم)**: العميل بيتصل بالسيرفر عبر منفذ TCP 21، وده بيُستخدم لإرسال الأوامر (مثل "أبعت ملف" أو "جيب ملف") واستقبال ردود السيرفر.
  2. **الاتصال الثاني (نقل البيانات)**: لما يكون في ملف هيتنقل، بيتم فتح اتصال جديد عبر منفذ TCP 20، وده مخصوص لنقل الملفات نفسها.
  3. **اتجاه النقل**: العميل ممكن يسحب (يحمل) ملفات من السيرفر أو يرفع (يبعت) ملفات للسيرفر.
- **يعني ايه؟**: زي لما تستخدم برنامج زي FileZilla لتحميل أو رفع ملفات من/إلى سيرفر معين.
