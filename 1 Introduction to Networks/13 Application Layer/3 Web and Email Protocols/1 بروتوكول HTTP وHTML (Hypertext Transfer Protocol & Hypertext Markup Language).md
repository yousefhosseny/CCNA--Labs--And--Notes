
### 1. **بروتوكول HTTP وHTML (Hypertext Transfer Protocol & Hypertext Markup Language)**:
- **الفكرة**: لما تكتب عنوان موقع (URL) في المتصفح (مثل http://www.cisco.com/index.html)، المتصفح بيتصل بخادم الويب عشان يجيب الصفحة.
- **الخطوات**:
  1. **تفسير الـURL**:
     - **http**: البروتوكول المستخدم.
     - **www.cisco.com**: اسم الخادم.
     - **index.html**: اسم الملف المطلوب.
  2. **ترجمة اسم الخادم**:
     - المتصفح بيتصل بخادم أسماء (DNS) عشان يحول www.cisco.com إلى عنوان IP (رقم زي 198.133.219.25).
     - المتصفح بيبعت طلب **GET** للخادم عشان يطلب ملف index.html.
  3. **رد الخادم**:
     - الخادم بيبعت كود HTML بتاع الصفحة للمتصفح.
  4. **عرض الصفحة**:
     - المتصفح بيقرأ كود HTML ويعرض الصفحة على شاشتك (نصوص، صور، إلخ).
- **مثال بسيط**: لما تفتح موقع، المتصفح بيطلب الصفحة من الخادم، والخادم بيرد بالمحتوى اللي تشوفه.
