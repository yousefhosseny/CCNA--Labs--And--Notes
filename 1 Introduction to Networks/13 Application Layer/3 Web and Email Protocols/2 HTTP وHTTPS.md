
### 2. **HTTP وHTTPS**:
- **HTTP**: بروتوكول يحدد طريقة التواصل بين المتصفح والخادم. فيه 3 أنواع رسائل رئيسية:
  - **GET**: المتصفح بيطلب بيانات (مثل صفحة ويب).
  - **POST**: المتصفح بيبعت بيانات للخادم (مثل إدخال بيانات في نموذج).
  - **PUT**: بيبعت ملفات أو موارد للخادم (مثل رفع صورة).
- **ملاحظة**: HTTP مش آمن، عشان كده بنستخدم **HTTPS** (النسخة الآمنة المشفرة) للتواصل عبر الإنترنت.
- **مثال بسيط**: لما تملّي نموذج على موقع، بتستخدم POST عشان تبعت البيانات، وHTTPS عشان تحميها.
