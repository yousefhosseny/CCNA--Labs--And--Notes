### شرح القسم بأبسط طريقة ممكنة:

القسم ده بيتكلم عن **كيفية ضبط عناوين IP** على الأجهزة النهائية (زي الكمبيوتر) وعلى السويتشات في شبكة Cisco، سواء يدويًا أو تلقائيًا. هنا الشرح بطريقة مبسطة:

---

#### 1. **ضبط عنوان IP يدويًا للأجهزة النهائية (Manual IP Address Configuration for End Devices):**
- الأجهزة النهائية (زي الكمبيوتر) لازم يكون ليها **عنوان IP** عشان تتواصل مع أجهزة تانية في الشبكة.
- فيه طريقتين لضبط عنوان IP: يدويًا أو تلقائيًا باستخدام **DHCP**.
- **لضبط عنوان IPv4 يدويًا على كمبيوتر ويندوز:**
  1. افتح **Control Panel** → **Network and Sharing Center** → **Change adapter settings**.
  2. اضغط كليك يمين على الشبكة (مثل Ethernet) واختار **Properties**.
  3. اختار **Internet Protocol Version 4 (TCP/IPv4)** واضغط **Properties**.
  4. ادخل يدويًا:
     - **عنوان IP** (مثل `192.168.1.10`).
     - **Subnet Mask** (مثل `255.255.255.0`).
     - **Default Gateway** (عنوان الروتر، مثل `192.168.1.1`).
  5. اضغط **OK** لحفظ الإعدادات.
- **ملاحظة:** ضبط IPv6 مشابه، بس بتستخدم عناوين سداسية (Hexadecimal) زي `2001:db8::1`.
