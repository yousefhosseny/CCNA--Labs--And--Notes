
#### 2. **ضبط عنوان IP تلقائيًا للأجهزة النهائية (Automatic IP Address Configuration for End Devices):**
- **DHCP** (Dynamic Host Configuration Protocol) هو نظام بيوزّع عناوين IP تلقائيًا للأجهزة.
- معظم الأجهزة النهائية بتكون مفعّلة لاستخدام DHCP افتراضيًا.
- **لضبط DHCP على ويندوز:**
  1. افتح **Control Panel** → **Network and Sharing Center** → **Change adapter settings**.
  2. اضغط كليك يمين على الشبكة واختار **Properties**.
  3. اختار **Internet Protocol Version 4 (TCP/IPv4)** واضغط **Properties**.
  4. اختار:
     - **Obtain an IP address automatically**.
     - **Obtain DNS server address automatically**.
  5. اضغط **OK** لحفظ الإعدادات.
- **ملاحظة:** بالنسبة لـ IPv6، بيستخدم **DHCPv6** أو **SLAAC** (Stateless Address Autoconfiguration) لتوزيع العناوين تلقائيًا.
