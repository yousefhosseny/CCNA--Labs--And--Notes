
#### 2. **تعديل الإعدادات الحالية (Alter the Running Configurations):**
لو عملت تغييرات في الـ running-config وما كانتش ناجحة:
- **لو لسه ما حفظتش التغييرات في startup-config:**
  - ممكن تمسح الأوامر الغلط واحد واحد يدويًا.
  - أو تعمل إعادة تشغيل للجهاز باستخدام:
    ```
    reload
    ```
    (تحذير: ده هيوقف الجهاز مؤقتًا، يعني الشبكة هتتوقف للحظات).

![[Pasted image 20250718171905.png]]

- **لو حفظت التغييرات الغلط في startup-config:**
  - امسح ملف الـ startup-config باستخدام:
    ```
    erase startup-config
    ```
  - بعد كده، اعمل `reload` عشان تمسح الـ running-config من الـ RAM.
  - (ملاحظة: بعد المسح، الجهاز هيرجع للإعدادات الافتراضية).
