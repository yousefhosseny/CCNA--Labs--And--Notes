
#### 5. **رسائل التحذير (Banner Messages):**
- الـ **Banner** هي رسالة تحذيرية بتظهر لأي حد يحاول يدخل الجهاز.
- بتستخدم عشان تحذر الأشخاص غير المصرح لهم من محاولة الوصول.
- **الأمر:**
  ```
  banner motd # [الرسالة] #
  ```
  - الرمز `#` هو **المحدد (delimiter)**، بتحطه قبل وبعد الرسالة.
  - مثال:
    ```
    banner motd # Authorized Access Only! #
    ```
- الرسالة هتظهر لأي حد يحاول يدخل الجهاز.
