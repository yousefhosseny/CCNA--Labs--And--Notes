
#### 2. **أوضاع التهيئة (Configuration Modes):**
لما تيجي تعدل إعدادات الجهاز، بتدخل أوضاع تهيئة مختلفة:

- **Global Configuration Mode:**
  - بيستخدم لتعديل إعدادات الجهاز ككل (زي إعدادات عامة).
  - الـ prompt بيكون: `(config)#` (مثال: `Switch(config)#`).

![[Pasted image 20250718163746.png]]


- **Line Configuration Mode:**
  - بيستخدم لتهيئة طرق الوصول للجهاز (زي Console، SSH، Telnet، أو AUX).
  - الـ prompt بيكون: `(config-line)#`.

![[Pasted image 20250718163804.png]]


- **Interface Configuration Mode:**
  - بيستخدم لتهيئة واجهات الجهاز (زي منافذ السويتش أو واجهات الروتر).
  - الـ prompt بيكون: `(config-if)#`.

![[Pasted image 20250718163819.png]]
