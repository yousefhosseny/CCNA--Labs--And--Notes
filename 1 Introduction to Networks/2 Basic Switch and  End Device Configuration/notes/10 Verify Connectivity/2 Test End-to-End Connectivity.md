#### 2. **فيديو: Test End-to-End Connectivity**
الفيديو ده بيشرح إزاي تختبر الاتصال بين الأجهزة (سويتشات وكمبيوترات) باستخدام أمر `ping`:

- **أمر ping:**
  - بيستخدم عشان تتأكد إن جهازين (زي كمبيوتر وسويتش، أو سويتش وسويتش) بيتواصلوا مع بعض.
  - الأمر بيبعت حزمة بيانات صغيرة لجهاز تاني ويستنى رد (Reply) عشان يتأكد إن الاتصال شغال.
  - مثال:
    ```
    ping 192.168.1.2
    ```
    (ده بيختبر الاتصال مع جهاز بعنوان IP `192.168.1.2`).

- **الخطوات:**
  1. ادخل وضع **Privileged EXEC** على السويتش أو الكمبيوتر:
     ```
     enable
     ```
  2. نفّذ أمر `ping` مع عنوان IP للجهاز الثاني:
     - مثال: لو عايز تختبر الاتصال من سويتش لكمبيوتر:
       ```
       ping 192.168.1.10
       ```
  3. لو الـ ping نجح، هتشوف ردود (Replies) في الشاشة، زي:
     ```
     Reply from 192.168.1.10: bytes=32 time=1ms TTL=128
     ```
     لو فشل، هيظهر رسالة زي "Request timed out".

- **الهدف:**
  - اختبار الاتصال بين:
    - سويتش وسويتش.
    - كمبيوتر وكمبيوتر.
    - كمبيوتر وسويتش.
  - التأكد إن الشبكة شغالة كويس من طرف لطرف (End-to-End Connectivity).
