
#### 2. **فحص صيغة الأوامر (IOS Command Syntax Check):**
- كل أمر ليه **صيغة معينة** لازم تتبعها. عشان تعرف الصيغة الصحيحة:
  - **النصوص الغامقة (Boldface):** دي الأوامر أو الكلمات المفتاحية اللي لازم تكتبها زي ما هي.
  - **النصوص المائلة (Italics):** دي القيم اللي أنت بتحددها (زي عنوان IP).
  - **الأقواس المربعة [x]:** تعني إن العنصر اختياري (مش لازم تكتبه).
  - **الأقواس المعقوفة {x}:** تعني إن العنصر إجباري (لازم تكتبه).
  - **الخط العمودي | :** يعني لازم تختار واحد من الخيارات (مثل [x | y]).

- **أمثلة:**
  - `ping 10.10.10.5`: الأمر `ping` والقيمة `10.10.10.5`.
  - `traceroute 192.168.254.254`: الأمر `traceroute` والقيمة `192.168.254.254`.
