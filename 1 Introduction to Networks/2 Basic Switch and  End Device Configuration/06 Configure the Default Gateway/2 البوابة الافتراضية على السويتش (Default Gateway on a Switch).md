
#### 2. **البوابة الافتراضية على السويتش (Default Gateway on a Switch):**
- السويتش محتاج **Default Gateway** عشان تتحكم فيه عن بُعد (مثل عبر SSH أو Telnet) من شبكة تانية.
- **الأمر لضبط البوابة الافتراضية:**
  - في وضع Global Configuration:
    ```
    ip default-gateway [عنوان IP]
    ```
    مثال: `ip default-gateway 192.168.10.1` (عنوان واجهة الروتر).
- **ليه مهم؟**
  - بدون البوابة الافتراضية، السويتش مش هيقدر يرد على طلبات من شبكات خارجية.

- **ملاحظة:** فيه مشكلة في الرسم التوضيحي (Graphic) في القسم ده، بس المعلومات الأساسية صحيحة.
