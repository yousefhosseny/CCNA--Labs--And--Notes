
### 4. **تفعيل SSH (Enable SSH)**:
- **الفكرة**: SSH (Secure Shell) هو بروتوكول آمن للوصول للأجهزة عن بُعد (بديل لـ Telnet الغير آمن).
- **خطوات تفعيل SSH على جهاز Cisco**:
  1. **غيّر اسم الجهاز**: اختار اسم مميز للجهاز بأمر `hostname`.
  2. **حدد اسم النطاق**: استخدم أمر `ip domain-name` (مثل `example.com`).
  3. **ولّد مفتاح تشفير**: استخدم أمر `crypto key generate rsa` مع حجم مفتاح (يفضل 1024 بت على الأقل) عشان تشفر الاتصال.
  4. **إنشاء مستخدم محلي**: استخدم أمر `username` عشان تسجل اسم مستخدم وكلمة سر في قاعدة البيانات المحلية.
  5. **تفعيل المصادقة المحلية**: استخدم أمر `login local` للتحقق من المستخدم.
  6. **السماح بـ SSH**: استخدم أمر `transport input ssh` عشان تسمح بالاتصال عبر SSH.
- **مثال بسيط**: زي ما تستخدم تطبيق آمن زي WhatsApp بدل الرسايل العادية، SSH بيحمي الاتصال بينك وبين الراوتر.
