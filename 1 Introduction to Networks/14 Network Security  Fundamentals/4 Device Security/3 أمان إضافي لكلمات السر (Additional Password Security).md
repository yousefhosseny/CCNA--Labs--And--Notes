
### 3. **أمان إضافي لكلمات السر (Additional Password Security)**:
- **إجراءات لتأمين كلمات السر على أجهزة Cisco**:
  1. **تشفير كلمات السر**: استخدم أمر `service password-encryption` عشان تشفر كل كلمات السر النصية.
  2. **حد أدنى لطول كلمة السر**: استخدم أمر `security passwords min-length` عشان تحدد أقل طول لكلمة السر.
  3. **منع هجمات التخمين**: استخدم أمر `login block-for` عشان تقفل الجهاز بعد عدد معين من المحاولات الغلط.
  4. **إيقاف الجلسات الخاملة**: استخدم أمر `exec-timeout` عشان تقفل الوصول لو الجهاز بقى خامل فترة معينة.
- **مثال بسيط**: لو حد حاول يخمن كلمة السر 5 مرات غلط، الجهاز بيقفل مؤقتاً عشان يمنعه.
