#### **نموذج TCP/IP (4 طبقات):**
   - **طبقة التطبيقات**: بتعرض البيانات للمستخدم وبتدير التواصل (زي HTTP للمتصفحات).
   - **طبقة النقل**: بتدير التواصل بين الأجهزة عبر شبكات مختلفة (زي TCP).
   - **طبقة الإنترنت**: بتحدد أحسن طريق للبيانات (زي IP).
   - **طبقة الوصول للشبكة**: بتدير الأجهزة والوسائط اللي بتشكل الشبكة (زي Ethernet أو الواي فاي).
