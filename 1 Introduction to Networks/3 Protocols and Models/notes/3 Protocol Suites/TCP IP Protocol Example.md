 #### **مثال على مجموعة TCP/IP:**
   - مجموعة TCP/IP بتشتغل على طبقات مختلفة:
     - **طبقة التطبيقات**: زي بروتوكول HTTP (اللي بيخليك تتصفح مواقع الإنترنت).
     - **طبقة النقل**: زي TCP (بيدير إرسال البيانات بطريقة موثوقة).
     - **طبقة الإنترنت**: زي IP (بيحدد إزاي البيانات توصل من جهاز للتاني).
     - **طبقة الوصول للشبكة**: زي Ethernet أو WLAN (الواي فاي)، بتوصل البيانات داخل الشبكة المحلية.
