### 2. Port Numbers

- افتح Command Prompt في أي PC وجرب أوامر:
    
    - `netstat` (في Packet Tracer يظهر لك Connections موجودة).
        
    - راقب الـ Source Port و Destination Port.
        
- اربط Client مع Server (مثلاً HTTP Client مع Web Server).
    
    - شوف إن الـ Destination Port = 80 والـ Source Port بيتغير دايناميك من الـ Client.
