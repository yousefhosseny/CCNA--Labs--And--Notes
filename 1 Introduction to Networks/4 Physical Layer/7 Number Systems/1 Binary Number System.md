النظام الثنائي (Binary Number System) وعلاقته بعناوين IPv4:

### 1. **ما هو النظام الثنائي؟**
- النظام الثنائي يستخدم فقط الرقمين **0 و1**، ويُطلق على كل رقم فيه اسم **بت (bit)**.
- بينما النظام العشري (اللي بنستخدمه يوميًا) يحتوي على الأرقام من **0 إلى 9**.
- الأجهزة مثل أجهزة الراوتر والكمبيوترات بتستخدم النظام الثنائي للتعرف على بعضها من خلال عناوين الـ IP.

### 2. **عناوين IPv4**:
- عنوان IPv4 هو عبارة عن سلسلة مكونة من **32 بت** (يعني 32 خانة من 0 أو 1).
- الـ 32 بت مقسمين إلى **4 أجزاء**، كل جزء اسمه **أوكتت (octet)**، وكل أوكتت فيه **8 بت** (يعني 8 خانات).
- عشان يكون سهل على البشر، بنحول الأوكتت من النظام الثنائي إلى النظام العشري، ونفصل بينهم بنقطة (مثال: 192.168.11.10).

### 3. **التحويل من النظام الثنائي إلى العشري**:
- كل أوكتت (8 بت) بيتم تحويله لوحده.
- كل بت في الأوكتت ليه قيمة حسب موقعه (من اليمين لليسار):
  - الموقع 0 = 1
  - الموقع 1 = 2
  - الموقع 2 = 4
  - الموقع 3 = 8
  - الموقع 4 = 16
  - الموقع 5 = 32
  - الموقع 6 = 64
  - الموقع 7 = 128
- لو البت = 1، بنضرب قيمة الموقع في 1. لو البت = 0، بنضربها في 0.
- بعد كده نجمع كل القيم.

**مثال**: تحويل **11000000** إلى عشري:
- 1 × 128 = 128
- 1 × 64 = 64
- 0 × 32 = 0
- 0 × 16 = 0
- 0 × 8 = 0
- 0 × 4 = 0
- 0 × 2 = 0
- 0 × 1 = 0
- الجمع: 128 + 64 + 0 + 0 + 0 + 0 + 0 + 0 = **192**

**مثال كامل**: تحويل عنوان IPv4 ثنائي **11000000.10101000.00001011.00001010** إلى عشري:
- الأوكتت الأول (11000000) = 128 + 64 = **192**
- الأوكتت الثاني (10101000) = 128 + 32 + 8 = **168**
- الأوكتت الثالث (00001011) = 8 + 2 + 1 = **11**
- الأوكتت الرابع (00001010) = 8 + 2 = **10**
- الناتج: **192.168.11.10**

---

### 4. **التحويل من النظام العشري إلى الثنائي**:
- عشان نحول رقم عشري (مثل 168) إلى ثنائي، بنستخدم جدول القيم (128, 64, 32, 16, 8, 4, 2, 1).
- بنبص إذا الرقم أكبر من أو يساوي القيمة في الموقع:
  - لو نعم: نكتب **1**، ونطرح القيمة من الرقم، ونكمل.
  - لو لا: نكتب **0**، ونروح للقيمة اللي بعدها.

**مثال**: تحويل **168** إلى ثنائي:
- 168 ≥ 128؟ نعم → نكتب 1، نطرح 128 (168 - 128 = 40)
- 40 ≥ 64؟ لا → نكتب 0
- 40 ≥ 32؟ نعم → نكتب 1، نطرح 32 (40 - 32 = 8)
- 8 ≥ 16؟ لا → نكتب 0
- 8 ≥ 8؟ نعم → نكتب 1، نطرح 8 (8 - 8 = 0)
- 0 ≥ 4؟ لا → نكتب 0
- 0 ≥ 2؟ لا → نكتب 0
- 0 ≥ 1؟ لا → نكتب 0
- الناتج: **10101000**

---

### 5. **ليه النظام الثنائي مهم؟**
- الأجهزة (مثل الراوتر والكمبيوترات) بتفهم بس النظام الثنائي (0 و1).
- لكن البشر بيستخدموا النظام العشري عشان أسهل للقراءة والكتابة.
- لازم تفهم التحويل بين النظامين عشان تفهم إزاي عناوين الـ IPv4 بتتعامل مع الأجهزة.

---

### خلاصة:
- النظام الثنائي بيستخدم 0 و1، والعشري من 0 لـ 9.
- عنوان IPv4 هو 32 بت مقسمة لـ 4 أوكتت، كل أوكتت 8 بت.
- التحويل من ثنائي لعشري: نضرب كل بت في قيمة موقعه ونجمع.
- التحويل من عشري لثنائي: نقارن الرقم مع القيم (128, 64, ...) ونكتب 1 أو 0.
- الأجهزة بتستخدم الثنائي، والبشر بيستخدموا العشري لسهولة القراءة.
