
### 5. **EUI-64 مقابل التوليد العشوائي**:
- لو الجهاز بيستخدم **SLAAC** أو **SLAAC مع Stateless DHCPv6**، لازم يولّد **Interface ID** بنفسه.
- فيه طريقتين لتوليد الـ Interface ID:
  - **EUI-64**:
    - بيستخدم عنوان MAC (48 بت) بتاع الجهاز.
    - بيضيف قيمة `fffe` في النص (بيصير 64 بت).
    - بيعكس البت السابع في عنوان MAC (من 0 لـ 1 أو العكس).
    - مثال: عنوان MAC هو `fc:99:47:75:ce:e0` بيصير Interface ID: `fe:99:47:ff:fe:75:ce:e0`.
  - **توليد عشوائي**:
    - بعض الأنظمة (زي Windows من Vista وطالع) بتولّد Interface ID عشوائي بدل EUI-64.
    - مثال من إخراج أمر `ipconfig`:
      ```
      IPv6 Address: 2001:db8:acad:1:50a5:8a35:a5bb:66e1
      Link-local IPv6 Address: fe80::50a5:8a35:a5bb:66e1
      Default Gateway: fe80::1
      ```
